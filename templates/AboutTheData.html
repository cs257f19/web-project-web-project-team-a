Carleton Course Facilitator

An open source program designed by and for Carleton students so that they can better select and plan their course load.

About the Data

This data was scraped off of a pdf version of the Carleton Course Catalogue using python script. The following variables were collected about each class:

Department - Which academic department teaches the class

Class ID Number - The ID code associated with the class, can be used to look it up on the Hub

Name - The title of the class

Prerequisites - Classes that must have been taken in order to take the selected class

Liberal Arts Requirements Fulfilled - These are the graduation requirements that the selected class fulfills

Limitations

While we initially contacted the Registrar's Office in hopes they would give us the data in .csv format, they declined. As a result, we had to use a course catalogue from the previous year. Some data may be outdated. Additionally, we were unable to grab data regarding which teachers were teaching each section of a class.

This site was made by Ben Preiss, Cam Brown, Tony Ngo with the goal of streamlining the class selection process and helping other students better search for classes that fit their interests. Last Updated: 10/17/2019
