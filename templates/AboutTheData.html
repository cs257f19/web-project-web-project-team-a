Carleton Course Facilitator

An open source program designed by and for Carleton students so that they can better select and plan their course load.

About the Data

This data was scraped off of Carleton Enroll page using python script. The following variables were collected about each class:

Department - Which academic department teaches the class

Class ID Number - The ID code associated with the class, can be used to look it up on the Hub

Name - The title of the class

Term - The term the course is offered

Liberal Arts Requirements Fulfilled - These are the graduation requirements that the selected class fulfills

Period - The class period which the class is available

Course Description - The description of the course

Limitations

While we initially contacted the Registrar's Office in hopes they would give us the data in .csv format, they declined. As a result, we created a webcrawler that traversed through carleton enroll in order to create a database that contains all course and course information at Carleton for Fall 2019, Winter 2020, and Spring 2020. Due to different departments having irregularities in their course information in the course description section of the database, there are \regex words that appear in the description due to how the html was written on Enroll. Removing the regex words proved to be more difficult than anticipated. It was decided that while it does make some of the course desciptions difficult to read when it occurs, it does not have a drastically negative impact on our class queries and results.
